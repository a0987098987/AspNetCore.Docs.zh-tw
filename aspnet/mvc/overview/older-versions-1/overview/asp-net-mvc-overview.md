- <span data-ttu-id="db394-119">**模型**。</span><span class="sxs-lookup"><span data-stu-id="db394-119">**Models**.</span></span> <span data-ttu-id="db394-120">模型物件屬於實作應用程式資料網域的邏輯應用程式的組件。</span><span class="sxs-lookup"><span data-stu-id="db394-120">Model objects are the parts of the application that implement the logic for the application s data domain.</span></span> <span data-ttu-id="db394-121">通常，模型物件會擷取並儲存在資料庫中的模型狀態。</span><span class="sxs-lookup"><span data-stu-id="db394-121">Often, model objects retrieve and store model state in a database.</span></span> <span data-ttu-id="db394-122">例如，Product 物件可能會從資料庫擷取資訊、 運作，和回 SQL Server 中的 Products 資料表寫入的更新的資訊。</span><span class="sxs-lookup"><span data-stu-id="db394-122">For example, a Product object might retrieve information from a database, operate on it, and then write updated information back to a Products table in SQL Server.</span></span>
