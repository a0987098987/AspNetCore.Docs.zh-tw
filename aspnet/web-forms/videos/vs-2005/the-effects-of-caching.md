<a name="the-effects-of-caching"></a>快取的效果
====================
由 Eric Lee

現今幾乎所有 Web 應用程式都需要存取資料庫的資料，但常見的作業可能會影響實際應用程式的整體效能。 這段影片會說明 Visual Studio Team System 可以幫助您識別其中是否正在資料庫中，重複呼叫您的應用程式，並接著示範您如何能運用 ASP.NET 2.0 的資料來源快取功能以消除多餘的呼叫。
