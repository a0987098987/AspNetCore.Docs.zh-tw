<span data-ttu-id="ad612-105">*Web 伺服陣列*是一組兩個或多個 Web 伺服器 (或稱為「節點」)，用於裝載應用程式的多個執行個體。</span><span class="sxs-lookup"><span data-stu-id="ad612-105">A *web farm* is a group of two or more web servers (or *nodes*) that host multiple instances of an app.</span></span> <span data-ttu-id="ad612-106">當使用者的要求抵達 Web 伺服陣列時，「負載平衡器」會將要求分散到 Web 伺服陣列的節點。</span><span class="sxs-lookup"><span data-stu-id="ad612-106">When requests from users arrive to a web farm, a *load balancer* distributes the requests to the web farm's nodes.</span></span> <span data-ttu-id="ad612-107">Web 伺服陣列改善：</span><span class="sxs-lookup"><span data-stu-id="ad612-107">Web farms improve:</span></span>

<span data-ttu-id="ad612-115">本主題針對依賴共用資源的 Web 伺服陣列，說明其中所裝載 ASP.NET Core 應用程式的設定和相依性。</span><span class="sxs-lookup"><span data-stu-id="ad612-115">This topic describes configuration and dependencies for ASP.NET core apps hosted in a web farm that rely upon shared resources.</span></span>
